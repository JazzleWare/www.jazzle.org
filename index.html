Jazzle is a new parser for all versions of ECMAScript, itself written in ECMAScript.

It obsessively strives to be fully compatible with the ECMAScript specification, and yet, it meets (and in nearly all benchmarks, beats) many other similar parsers currently in production.

Its high speed, combined with its ability to catch the errors that won't get caught otherwise, makes it an excellent choice for integration with linters, IDE's, and a myriad of other ECMAScript code assessment tools. On an average 2012 laptop,
